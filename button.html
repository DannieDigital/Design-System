Buttons

Buttons are used as triggers for actions and events.

Primary

Primary call to action button.

Example:

Code:



<button href="#" class="btn-primary">Primary</button>

Default

Default button used throughout design system.

Example:

Code:



<button type="button" class="btn btn-outline-secondary">Default</button>

Accessibility

ARIA attributes provides extra context for screenreader users when buttons are used to make changes on the current page.
