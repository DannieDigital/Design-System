Navigation Bar

The navigation bar is used to navigate the website. Navigation bar shall do the following:

  • Present on all pages at the top.
  • Appear at all times while scrolling.
  • Logo link to the homepage.
  • Active page must be highlighted.

Example:

Danielle Beaty
  • Home
  • About
  • Contact
  • Resume

Code:


<nav class="navbar navbar-expand-lg bg-body-tertiary sticky-top">
<div class="container-fluid navcolor">
  <a class="navbar-brand" href="#">
    <img src="../images/logo.jpg" alt="Logo" width="98" height="72" class="d-inline-block align-text-center">
  Saman Kamran
    </a>
    <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav" aria-controls="navbarNav" aria-expanded="false" aria-label="Toggle navigation"
      <span class="navbar-toggler-icon"></span>
      </button>
    <div class="collapse navbar-collapse" id="navbarNav"
      <ul class="navbar-nav ms-auto">
        <li class="nav-item">
          <a class="nav-link active" aria-current="page" href="#">Home</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">About</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">Contact</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">Resume</a>
        </li>
      </ul>
    </div>
  </div>
</nav>
